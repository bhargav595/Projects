WebScrapping Online Novels

The project will parse the table of contents of the specified novel from koreanmtl.online website, scrap the data of each chapter of the novel and store the extracted data in local files. 

The project is implemented using Python with 
-libraries such as 
  1. Beautiful Soap - To parse teh HTML source code
  2. tkinter - To store the rextracted data to a local file
-Requests module - To send HTTP requests using Python
